# CircularApp - Evaluador de circularidad de productos

CircularApp es una herramienta educativa que analiza la sostenibilidad, reparabilidad y huella ambiental de tus dispositivos electrónicos. Su misión es empoderar a consumidores, empresas y profesionales para tomar decisiones informadas que reduzcan el impacto ambiental del sector tecnológico.

## Flujo

Para realizar el proyecto se ha realizado unos estilos globales de css en `css/index.css`, el cual hace un reinicio de estilos con `css/reset.css`. A partir de ahí, cada página depende de sus propios estilos, al igual que de su archivo _JavaScript_ si es que lo contiene.
